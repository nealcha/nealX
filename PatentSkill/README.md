# AIX

`AIX` is a personal repository for local-first AI workflow tools.

Current project:

- `patent-copilot-workflow/`: a reusable utility-model patent drafting workflow that turns local patent PDFs and optional research notes into a reviewed patent package.

Use [patent-copilot-workflow/README.md](patent-copilot-workflow/README.md) for the project overview and [patent-copilot-workflow/USAGE.md](patent-copilot-workflow/USAGE.md) for the simplest run path.
